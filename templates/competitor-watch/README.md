# Competitor Watch

Monitors competitor websites and feeds; produces a weekly briefing.

## Setup

Set COMPETITOR_URLS (and optionally COMPETITOR_FEEDS, KEYWORDS, OUTPUT_DIR) in `.env`.

## Schedule

Run weekly (e.g. cron `0 9 * * 1` for Monday 09:00).

## Output

Markdown briefing in OUTPUT_DIR or stdout.
